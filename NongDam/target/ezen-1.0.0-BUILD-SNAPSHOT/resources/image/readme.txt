	!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
	
	
	저장시에 )
	
	ex) myPage/main에서 최초로 저장되는 user테이블의 myProfile 이미지의 경로
	resources/image/myPage/파일명.png
	
	ex) 1:1거래게시판(tr)에서 글 작성시 최초로 저장되는 tr 테이블의 썸네일 이미지 경로
	resources/image/tr/파일명.png
	
	
	-------------------
	
	불러올 때 )
		servlet-context.xml에 의해
		
		<img src="${contextPath}/resources/image/..." />
		
		같은 경로로 불러오면 되도록 설정되어 있습니다.
